# Similarity Detection Tool

Web application that allows to compare your personal document (i.e. a thesis) against a number of resources (literature) and highlights similarities.

## Author

* Lukas Lebovitz

## Technology Stack used

* Spring Boot
* H2 & JPA
* React & Redux
* SockJS & Stomp
* Apache PDFBox
